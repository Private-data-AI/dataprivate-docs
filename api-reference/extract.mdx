## Overview

The `/extract` endpoint analyzes a text and replaces personally identifiable information (PII) with neutral placeholders like `[FIRST_NAME_1]` or `[EMAIL_1]`. You control which entity types to detect and the minimum confidence threshold.

**Your data stays yours.** The API processes the text and returns results ã no data is stored or retained server-side.

## Authentication

All requests require an API key passed in the `X-API-Key` header.

```bash
X-API-Key: your_api_key
```

<Note>
  Contact the Data Private team to obtain your API key.
</Note>

## Request

### Headers

| Name        | Value              | Required |
|-------------|--------------------|----------|
| `X-API-Key` | Your API key       | Yes      |
| `Content-Type` | `application/json` | Yes   |

### Body Parameters

<ParamField body="text" type="string" required>
  The raw text to analyze. Supports plain text from emails, chats, documents, support tickets, etc.
</ParamField>

<ParamField body="labels" type="string[]">
  List of entity types to detect. If omitted or empty, **all supported types** are searched.

  See the [Supported Entity Types](#supported-entity-types) section for the full list.

  ```json
  ["first_name", "last_name", "email", "phone_number"]
  ```
</ParamField>

<ParamField body="confidence_threshold" type="float" default="0.1">
  Minimum confidence score required to include an entity in the results. Range: `0.0` to `1.0`.

  - `0.1` ã broad detection (default), catches more entities with some false positives
  - `0.8` ã strict detection, only high-confidence entities
</ParamField>

## Response

<ResponseField name="text" type="string">
  The original text with all detected entities replaced by numbered placeholders.

  Example: `"Hello [FIRST_NAME_1] [LAST_NAME_1], your email [EMAIL_1] has been updated."`
</ResponseField>

<ResponseField name="entities" type="array">
  List of detected entities.

  <Expandable title="Entity object">
    <ResponseField name="value" type="string">
      The original text value of the detected entity.
    </ResponseField>
    <ResponseField name="type" type="string">
      The entity type (e.g. `first_name`, `email`, `iban`).
    </ResponseField>
    <ResponseField name="score" type="float">
      Confidence score between `0.0` and `1.0`.
    </ResponseField>
    <ResponseField name="start" type="integer">
      Start character index of the entity in the original text.
    </ResponseField>
    <ResponseField name="end" type="integer">
      End character index of the entity in the original text.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="duration" type="float">
  Processing time in milliseconds.
</ResponseField>


### Raise the confidence threshold

Reduce false positives by requiring a higher confidence score.

```python Python
response = requests.post(
    "https://private-data.ai/api/v1/public/extract",
    headers={"X-API-Key": "your_api_key"},
    json={
        "text": "Patient Jean Martin, born 12/03/1978, blood type A+.",
        "labels": ["first_name", "last_name", "date_of_birth", "blood_type"],
        "confidence_threshold": 0.85
    }
)
```

### Use the entity map to restore original values

The API returns detected entities with their positions. You can use this to build your own anonymization/deanonymization logic locally ã nothing leaves your system.

```python Python
import requests

text = "Invoice for Marie Curie, IBAN FR76 3000 6000 0112 3456 7890 189."

response = requests.post(
    "https://private-data.ai/api/v1/public/extract",
    headers={"X-API-Key": "your_api_key"},
    json={"text": text}
)

result = response.json()
anonymized_text = result["text"]

# Build a local mapping: placeholder -> original value
entity_map = {}
entity_counts = {}

for entity in result["entities"]:
    t = entity["type"].upper()
    entity_counts[t] = entity_counts.get(t, 0) + 1
    placeholder = f"[{t}_{entity_counts[t]}]"
    entity_map[placeholder] = entity["value"]

print("Anonymized:", anonymized_text)
print("Map:", entity_map)

# Restore original text
restored = anonymized_text
for placeholder, original in entity_map.items():
    restored = restored.replace(placeholder, original)

print("Restored:", restored)
```
