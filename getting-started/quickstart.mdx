## Premier appel

Ce snippet anonymise un texte et le désanonymise — sans LLM, pour tester que l'API répond correctement.

<CodeGroup>
```python Python
import dataprivate

dp = dataprivate.Client(api_key="dp_...")

text = "Contrat de Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr"

# Anonymise
result = dp.anonymize(text)
print(result["anonymized_text"])
# → "Contrat de [PERSON_1], IBAN: [IBAN_1], email: [EMAIL_1]"

# Désanonymise
original = dp.deanonymize(result["anonymized_text"], result["tokens"])
print(original)
# → "Contrat de Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr"
```

```javascript JavaScript
import DataPrivate from 'dataprivate';

const dp = new DataPrivate({ apiKey: 'dp_...' });

const text = 'Contrat de Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr';

// Anonymise
const result = await dp.anonymize(text);
console.log(result.anonymizedText);
// → 'Contrat de [PERSON_1], IBAN: [IBAN_1], email: [EMAIL_1]'

// Désanonymise
const original = await dp.deanonymize(result.anonymizedText, result.tokens);
console.log(original);
// → 'Contrat de Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr'
```

```bash curl
# Anonymise
curl -X POST https://private-data.ai/api/v1/anonymize \
  -H "Authorization: Bearer dp_..." \
  -H "Content-Type: application/json" \
  -d '{"text": "Contrat de Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr"}'

# Désanonymise
curl -X POST https://private-data.ai/api/v1/deanonymize \
  -H "Authorization: Bearer dp_..." \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Contrat de [PERSON_1], IBAN: [IBAN_1], email: [EMAIL_1]",
    "tokens": {"PERSON_1": "Jean Dupont", "IBAN_1": "FR76 3000 6000 0112 3456 7890", "EMAIL_1": "jean.dupont@acme.fr"}
  }'
```
</CodeGroup>

## Appel complet avec un LLM

Une fois l'API validée, voici le pattern complet avec OpenAI :

<CodeGroup>
```python Python
import dataprivate
import openai

dp = dataprivate.Client(api_key="dp_...")

user_message = "Résume ce contrat pour Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890"

# 1. Anonymise avant envoi
anon = dp.anonymize(user_message)

# 2. Envoie le texte propre au LLM
response = openai.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": anon["anonymized_text"]}]
)

# 3. Restaure les valeurs réelles dans la réponse
result = dp.deanonymize(response.choices[0].message.content, anon["tokens"])
print(result)
```

```javascript JavaScript
import DataPrivate from 'dataprivate';
import OpenAI from 'openai';

const dp = new DataPrivate({ apiKey: 'dp_...' });
const openai = new OpenAI();

const userMessage = 'Résume ce contrat pour Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890';

const anon = await dp.anonymize(userMessage);

const response = await openai.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: anon.anonymizedText }]
});

const result = await dp.deanonymize(response.choices[0].message.content, anon.tokens);
console.log(result);
```
</CodeGroup>

<Note>
  Les tokens générés sans `session_id` sont valides uniquement pour l'appel en cours. Pour des conversations multi-turn, utilise un `session_id` — les tokens restent cohérents pendant 60 minutes.
</Note>
