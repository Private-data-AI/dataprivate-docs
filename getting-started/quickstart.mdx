## First call

This snippet detects PII in a text and prints the anonymized result and entity map.

<CodeGroup>
```python Python
import requests

response = requests.post(
    "https://private-data.ai/api/v1/public/extract",
    headers={"X-API-Key": "your_api_key"},
    json={
        "text": "Contract for Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr"
    }
)

result = response.json()
print(result["text"])
# → "Contract for [FIRST_NAME_1] [LAST_NAME_1], IBAN: [IBAN_1], email: [EMAIL_1]"

print(result["entities"])
# → [{"value": "Jean", "type": "first_name", "score": 0.99, "start": 13, "end": 17}, ...]
```

```javascript JavaScript
const response = await fetch("https://private-data.ai/api/v1/public/extract", {
  method: "POST",
  headers: {
    "X-API-Key": "your_api_key",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    text: "Contract for Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr"
  })
});

const result = await response.json();
console.log(result.text);
// → "Contract for [FIRST_NAME_1] [LAST_NAME_1], IBAN: [IBAN_1], email: [EMAIL_1]"
```

```bash cURL
curl -X POST https://private-data.ai/api/v1/public/extract \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"text": "Contract for Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890, email: jean.dupont@acme.fr"}'
```

## Full pattern with an LLM

Once the API is validated, here's the complete pattern with OpenAI:

```python Python
import requests
import openai

# 1. Detect and anonymize
result = requests.post(
    "https://private-data.ai/api/v1/public/extract",
    headers={"X-API-Key": "your_api_key"},
    json={"text": "Summarize this contract for Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890"}
).json()

anonymized_text = result["text"]
entities = result["entities"]

# 2. Send clean text to LLM
client = openai.OpenAI(api_key="sk-...")
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {
            "role": "system",
            "content": "Preserve exactly all tokens in the format [TYPE_N] without any modification."
        },
        {"role": "user", "content": anonymized_text}
    ]
)

llm_response = response.choices[0].message.content

# 3. Restore values locally — build the map from entities
entity_map = {}
counters = {}
for entity in entities:
    t = entity["type"].upper()
    counters[t] = counters.get(t, 0) + 1
    entity_map[f"[{t}_{counters[t]}]"] = entity["value"]

restored = llm_response
for placeholder, original in entity_map.items():
    restored = restored.replace(placeholder, original)

print(restored)
```

<Note>
  Deanonymization happens entirely in your code. Data Private never receives the original values back — the mapping lives only in your memory.
</Note>
