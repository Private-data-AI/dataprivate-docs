## Pattern de base

<CodeGroup>
```python Python
import dataprivate
import openai

dp = dataprivate.Client(api_key="dp_...")
client = openai.OpenAI(api_key="sk-...")

def llm_call_safe(user_message: str) -> str:
    # 1. Anonymise
    anon = dp.anonymize(user_message)

    # 2. Appel LLM avec texte propre
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {
                "role": "system",
                "content": "Preserve exactly all tokens in the format [TYPE_N] without any modification."
            },
            {"role": "user", "content": anon["anonymized_text"]}
        ]
    )

    # 3. Restaure les valeurs dans la réponse
    return dp.deanonymize(
        text=response.choices[0].message.content,
        tokens=anon["tokens"]
    )

result = llm_call_safe(
    "Résume ce contrat pour Jean Dupont, IBAN: FR76 3000 6000 0112 3456 7890"
)
print(result)
```

## Avec historique de conversation (session_id)

Utilise un `session_id` pour que les tokens soient cohérents sur toute la conversation. Data Private gère le dictionnaire — tu n'as pas à passer `tokens` au `/deanonymize`.

<CodeGroup>
```python Python
import uuid
import dataprivate
import openai

dp = dataprivate.Client(api_key="dp_...")
client = openai.OpenAI(api_key="sk-...")

SYSTEM_PROMPT = "Preserve exactly all tokens in the format [TYPE_N] without any modification."

def chat_session():
    messages = [{"role": "system", "content": SYSTEM_PROMPT}]
    session_id = f"ses_{uuid.uuid4().hex[:12]}"  # valide 60 min

    while True:
        user_input = input("You: ")
        if user_input.lower() == "exit":
            break

        # Anonymise avec session — tokens cohérents sur toute la conversation
        anon = dp.anonymize(user_input, session_id=session_id)
        messages.append({"role": "user", "content": anon["anonymized_text"]})

        response = client.chat.completions.create(model="gpt-4o", messages=messages)
        assistant_msg = response.choices[0].message.content
        messages.append({"role": "assistant", "content": assistant_msg})

        # Désanonymise via session — pas besoin de passer tokens
        restored = dp.deanonymize(text=assistant_msg, session_id=session_id)
        print(f"Assistant: {restored}")
```

```javascript JavaScript
import { randomUUID } from 'crypto';
import DataPrivate from 'dataprivate';
import OpenAI from 'openai';

const dp = new DataPrivate({ apiKey: 'dp_...' });
const openai = new OpenAI({ apiKey: 'sk-...' });

const SYSTEM_PROMPT = 'Preserve exactly all tokens in the format [TYPE_N] without any modification.';

async function chatSession(turns) {
  const messages = [{ role: 'system', content: SYSTEM_PROMPT }];
  const sessionId = `ses_${randomUUID().replace(/-/g, '').slice(0, 12)}`;

  for (const userInput of turns) {
    const anon = await dp.anonymize({ text: userInput, sessionId });
    messages.push({ role: 'user', content: anon.anonymizedText });

    const response = await openai.chat.completions.create({ model: 'gpt-4o', messages });
    const assistantMsg = response.choices[0].message.content;
    messages.push({ role: 'assistant', content: assistantMsg });

    const restored = await dp.deanonymize({ text: assistantMsg, sessionId });
    console.log(`Assistant: ${restored}`);
  }
}
```

## Avec streaming

<Warning>
  Accumule le stream complet avant de désanonymiser. Un token `[PERSON_1]` peut être coupé sur plusieurs chunks.
</Warning>

```python Python
import dataprivate
import openai

dp = dataprivate.Client(api_key="dp_...")
client = openai.OpenAI(api_key="sk-...")

user_message = "Analyse le dossier de Marie Martin, née le 03/07/1990"
anon = dp.anonymize(user_message)

full_response = ""
with client.chat.completions.stream(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "Preserve exactly all tokens in the format [TYPE_N]."},
        {"role": "user", "content": anon["anonymized_text"]}
    ]
) as stream:
    for chunk in stream:
        delta = chunk.choices[0].delta.content or ""
        full_response += delta
        print(delta, end="", flush=True)

# Désanonymise après le stream complet
restored = dp.deanonymize(text=full_response, tokens=anon["tokens"])
print(f"\n\nRésultat final : {restored}")
```

## FAQ

<AccordionGroup>
  <Accordion title="Le LLM modifie parfois mes tokens, que faire ?">
    Ajoute cette instruction dans ton system prompt :
    ```
    Preserve exactly all tokens in the format [TYPE_N] without any modification.
    Do not translate, lowercase, or reformat tokens like [PERSON_1] or [IBAN_1].
    ```
    Si le problème persiste, utilise `gpt-4o` plutôt que `gpt-3.5-turbo` — les modèles plus petits suivent moins bien les instructions de format.
  </Accordion>

  <Accordion title="Dois-je anonymiser le system prompt ?">
    Non, sauf si ton system prompt contient lui-même des PII. Dans ce cas, anonymise-le séparément avec le même `session_id` et les tokens seront automatiquement cohérents.
  </Accordion>

  <Accordion title="Combien de temps dure une session ?">
    60 minutes à partir du premier appel `/anonymize` avec ce `session_id`. Une session expirée retourne une erreur `404 session_not_found` — génère un nouveau `session_id` et redemarre.
  </Accordion>

  <Accordion title="Quelle latence ajoute Data Private ?">
    Typiquement 50–150ms par appel `/anonymize` ou `/deanonymize`. Pour minimiser l'impact, parallélise l'appel `/anonymize` avec la préparation de ta requête si possible.
  </Accordion>
</AccordionGroup>
